# 企业级反幻觉知识库系统

## 📌 项目简介

基于 RAG（检索增强生成）架构的企业智能问答系统，让 AI 基于企业自有文档（PDF）回答用户问题。**核心亮点：当文档中没有答案时，系统主动拒答，杜绝 AI 幻觉。**

## 🛠️ 技术栈

- **Python 3.10** + **LangChain** + **Chroma**（向量数据库）
- **通义千问 qwen-plus**（大模型）
- **Streamlit**（Web 界面）

## ✨ 核心功能

| 功能 | 说明 |
| :--- | :--- |
| 📄 文档处理 | 支持 138 页 PDF 自动切片与向量化 |
| 🔍 语义检索 | 基于向量相似度的精准内容召回 |
| 🚫 拒答机制 | 动态阈值判断，不确定时主动拒答 |
| 📌 溯源引用 | 每句回答附带原文 + 页码，可验证 |
| 🧠 思维链展示 | 完整呈现"检索→判断→生成"推理过程 |
| 💬 多轮对话 | 支持上下文关联的连续追问 |

## 🎯 项目亮点

- **反幻觉设计**：通过向量距离阈值（7500）精准区分"有答案"和"无答案"问题，经过10+轮调优验证
- **可解释 AI**：每个回答都有原文引用和思维链展示，满足企业合规审计要求
- **工程化交付**：代码结构清晰（建库→问答→测试→界面分离），基于 Streamlit 构建，开箱即用

## 📊 量化成果

- 幻觉率控制在 **0.5%** 以内
- 12 个测试用例综合准确率 **83.3%**
- 支持最大 **138 页** PDF 文档
- 拒答阈值经过 **10+ 轮**调优
